# Vikas Mishra Portfolio

Premium modern developer portfolio built with Next.js, React, Tailwind CSS, and Framer Motion.

## Stack

- Next.js App Router
- React
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
